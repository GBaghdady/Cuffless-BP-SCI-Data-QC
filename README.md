# Cuffless BP SCI Data QC

This repository contains the reproducible analysis pipeline used for cuffless blood pressure analysis in an SCI cohort.

---

## Run in Google Colab

Open the notebook directly in Colab using the link below:

**[Open in Colab](https://colab.research.google.com/github/GBaghdady/Cuffless-BP-SCI-Data-QC/blob/main/Signal_QC_Pipeline.ipynb)**

---

## How to Use

1. Open the notebook in Google Colab.
2. Upload the required dataset when prompted **OR** edit the path to load the sample dataset from the `data/` folder.
3. Run all cells from top to bottom.
4. Outputs and QC results will be generated automatically.

---

## Notes

This repository contains analysis code only.  
It does **not** provide clinical diagnosis.

---

## Data

The folder `data/` contains example input files required to run the QC pipeline.

These data are provided for **demonstration and reproducibility only**.  
No identifiable patient information is included.

### Files

- [`Example Data.xlsx`](data/Example%20Data.xlsx) — Sample dataset used to demonstrate the signal QC pipeline.

Real patient datasets are not publicly shared due to confidentiality.
